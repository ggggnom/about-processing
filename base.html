CАМОЕ ПРОСТОЕ И ОСНОВНОЕ

ВЗАИМОДЕЙСТВИЕ: ALERT, PROPMT, CONFIRM

ALERT

Используй, если хочешь сообщить что-то важное полюзователю или в качестве уведомления/предупреждения

Функция alert отобразит окно в браузере и приостановит дальнейшее выполнение скриптов до тех пор, пока пользователь не нажмёт кнопку «OK». Это небольшое окно с сообщением называется модальным окном.

Понятие модальное означает, что пользователь не может взаимодействовать с интерфейсом остальной части страницы, нажимать на другие кнопки и т.д. до тех пор, пока взаимодействует с окном. В данном случае — пока не будет нажата кнопка «OK».

alert(message);
alert("Hello");

PROMPT

Используй, если надо узнать возраст пользователя или другую мин. информацию о пользователе, которая может повлиять на его взаимодесвие с сайтом

Эта функция отобразит модальное окно с текстом, полем для ввода текста и кнопками OK/Отмена. Функция prompt принимает два аргумента: title и default.Первый аргумент отвечает за текст, отображаемый в модальном окне. Второй параметр необезателен. Он устанавливает начальное значение в поле для текста в окне. Пользователь может напечатать что-либо в поле ввода и нажать OK. Он также может отменить ввод нажатием на кнопку «Отмена» или нажав на клавишу Esc.

prompt(title, [default]);

let age = prompt('Сколько тебе лет?', 100);
alert(`Тебе ${age} лет!`); // Тебе 100 лет!

CONFIRM

Используй, если cледует получить ответ от пользователя "да" или "нет"

Функция confirm отображает модальное окно с текстом вопроса question и двумя кнопками: OK и Отмена. Результат true, если нажата кнопка OK. В других случаях — false.

result = confirm(question);

let isBoss = confirm("Ты здесь главный?");
alert( isBoss ); // true, если нажата OK

УСЛОВНЫЕ ОПЕРАТОРЫ : if, '?'

IF

Самый распространенный и простой оператор, который можно использовать для получения ответа да/нет, на выбор по критерия и условиям и многое другое

Оператор if(...) вычисляет условие в скобках и, если результат true, то выполняет блок кода.

let year = prompt('В каком году появилась спецификация ECMAScript-2015?', '');
if (year == 2015) alert( 'Вы правы!' );

RUN CODE

В примере выше, условие — это простая проверка на равенство (year == 2015), но оно может быть и гораздо более сложным. Если мы хотим выполнить более одной инструкции, то нужно заключить блок кода в фигурные скобки:.

if (year == 2015) {
alert( "Правильно!" );
alert( "Вы такой умный!" );
}

ELSE

Else используй, если помимо одного условия, есть второе

Оператор if может содержать необязательный блок «else» («иначе»). Выполняется, когда условие ложно.

Например:

let year = prompt('В каком году появилась спецификация ECMAScript-2015?', '');
if (year == 2015){
alert( 'Да вы знаток!' );
} else {
alert( 'А вот и неправильно!' ); // любое значение, кроме 2015
}
